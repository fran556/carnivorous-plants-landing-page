# 🌿 Carnivorus Plants - Landing Page

A minimalist, high-end landing page for a carnivorous plant boutique. 
Inspired by the "Rooted Love" concept by [Shruti Parashar](https://dribbble.com/Shru-tea).

[Live Demo](#) | [Design Reference](https://dribbble.com/shots/26065751-Rooted-Love-Plant-Shop-Landing-Page)

---

## ✨ Key Features
* **Aesthetic UI:** Clean, airy interface focused on nature.
* **Plant Care Quiz:** Interactive section to find the perfect "predator" plant.
* **Responsive Design:** Fully optimized for mobile and desktop.
* **Custom Filtering:** Sort plants by trap type (Active, Passive, Semi-active).
